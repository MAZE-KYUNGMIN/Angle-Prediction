# 프로젝트 순서
1. 로민에서 제공받은 데이터를 1도씩 틀어 360도의 사진 저장.
2. 모델 성능을 높이기위해 이미지 증식 후 vgg16모델 전이학습.
3. 몇 블록을 동결해야 가장 높은 성능이 나오는지 확인.
